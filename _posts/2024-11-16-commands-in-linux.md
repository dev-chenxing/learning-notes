### pacman

Installing Packages using the Pacman

```bash
sudo pacman -S arduino-ide
```

Upgrading packages

```bash
sudo pacman -Suy
```

### 7z

Extract all files from an archive to the current directory with full paths

```bash
7z x arduino-ide_2.3.3_Linux_64bit.zip
```
